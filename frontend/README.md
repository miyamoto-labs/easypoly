# EasyPoly Frontend

Next.js 15 web application.

## Setup

```bash
npm install
cp .env.example .env.local
# Add NEXT_PUBLIC_SYNTHDATA_API_KEY
npm run dev
```

## Structure

```
frontend/
├── src/
│   ├── app/         # Next.js app router
│   ├── components/  # React components
│   └── lib/         # Utils, API clients
└── public/          # Static assets
```
